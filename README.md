# Network-Traffic-Predictor
This is the code and data set that I used to train a model to predict the amount of views a website will get and a system to alert people about unusual internet traffic. 
It uses Linear Regression to predict the views and Random Tree to classify if traffic is unusual or not.

## Technologies
- JuypterNotebook
- Python

## Setup
1. Download the Notebook
2. [Find the training data here(look for train1.csv)](https://www.kaggle.com/c/web-traffic-time-series-forecasting)
3. Upload both to something that can run the notebook like a google colab
4. Hit Run
